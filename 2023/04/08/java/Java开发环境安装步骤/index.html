Java开发环境安装步骤

一、安装JDK

1、运行jdk安装包

一直点击下一步，可以根据需求修改安装路径

2、配置环境变量

在我们编译程序时，我们需要用到javac这个编译工具，而javac是在java的安装目录下的，当我们在其他目录调用javac时，系统因为在当前目录找不到就会报错，为了方便我们以后编译程序，不要每次编译的时候就跑到java安装目录中去，所以我们来设置环境变量。

步骤：
右击计算机→属性→高级系统设置→高级→环境变量
（1）下面系统变量新建：
变量名固定，变量值为JDK的安装目录

（2)在系统变量中找到”Path” **
选中”Path“点击”编辑“选项

选择右边的“编辑文本”将引号里面的全部复制“%Java_Home%\bin;%Java_Home%\jre\bin;”**，点击确定。

三、安装Maven

1、下载Maven

官网地址
https://maven.apache.org/download.cgi
百度网盘直接下载
链接：https://pan.baidu.com/s/1MI_BoDGXt6wB1wzCsScS_w
提取码：root
将下载的maven解压到指定的目录

2、配置环境变量

右击计算机→属性→高级系统设置→高级→环境变量
编辑系统变量

编辑变量名为”Path”的值，将“%MAVEN_HOME%\bin;”添加到变量值前面

3、查看是否安装成功

启动CMD命令工具，输入：mvn -v，成功展示maven的版本号。

4、修改Maven仓库下载镜像

jar包下载慢的原因是服务器在国外，要翻墙才能下载，大约速度在0.9K/s的下载速度，所以我们安装Maven时，要及时的修改Maven下载的镜像地址，最好改为国内的下载镜像，例如阿里云中央仓库，华为云中央仓库。

打开setting.xml
在安装目录config文件夹中打开setting.xml,可以使用nodePad++进行编辑，百度网盘地址
链接：https://pan.baidu.com/s/1nHyqCnIksx5rEUE2cdRyJQ
提取码：root
在标签下添加阿里云镜像
